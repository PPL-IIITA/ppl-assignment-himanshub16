### Build instructions
**To execute Question 1**
```
make q1
./result
```

**To execute Question 2**
```
make q2
./result [k]
```

**To generate csv datasets**
```
make csv
./generate [girls] [boys] [gifts]
```

**To cleanup all the mess**
```
make clean
```
